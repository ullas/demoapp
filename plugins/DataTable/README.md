# CakePHP DataTable Plugin
DataTable is a cakephp plugin for [JQuery DataTables](http://datatables.net/).

# Requirements
* CakePHP >= 2.2
* PHP >= 5.4

# Documentation
Documentation is available on [wiki page](https://github.com/tigrang/cakephp-datatable/wiki)
